#吾家

##项目描述

一款为便利家庭亲人间的沟通交流以及丰富家庭生活的专有平台。家庭中任何角色成员均可在此平台简单地交流或是说出自己的心声，一些正能量的话语总会促使家庭更加和谐幸福；任一成员又可发起聚焦家庭其他成员的各种活动，比如聚会、任务分派、投票、红包发放等。只要是涉及家庭方面事情，均可考虑在此平台建立业务，比如丢失儿童的信息等。
